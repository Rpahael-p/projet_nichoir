# Librairies utilisées

Ce dossier contient tous les fichiers relatifs au librairie utilisée

## M5TimerCAM

Librairie utilisée pour controler le TimerCAM
le dossier [TimerCam](./TimerCam) contient:  
- les fichiers source corrigé de la librairie + de la camera du package M5Stack
- Un markdown explicatif du fonctionnement de la librairie

## PubSubClient

Librairie utilisée pour la communication MQTT de la TimerCam

## WiFi

Librairie utilisée pour la connexion wifi

## WebServer

Librairie utilisée pour le site de configuration via AP

## Preferences

Librairie utilisée pour stockée des information dans la mémoire non-volatile (NVS)

## Ticker

Librairie utilisée pour faire clignoter la led de la TimerCam en mode AP à l'aide d'un Timer

## Paho MQTT

Librairie utilisée pour le protocole MQTT en python  
[Paho MQTT](https://pypi.org/project/paho-mqtt/#client)
