# Raspberry

Ce dossier contient tous les fichiers en lien avec la partie Raspberry : 
- Base de donnée : MariaDB et SQLalchemy
- Broker MQTT : Mosquitto
- Interface Web : Flask

## Mise en place

#### 1)&nbsp;Installation des différents programmes.  
#### 2)&nbsp;Copie des dossiers DB et FLask vers votre raspberry.  
#### 3)&nbsp;Mise en place du broker mosquitto.  
Le fichier [Mosquitto Setup](mosquitto_setup.md) explique la configuration nécessaire du broker mosquitto
#### 4)&nbsp;Mise en place de la base de donnée, du listener mqtt et de l'interface Flask.
Le fichier [Project Setup](project_setup.md) explique la configuration nécessaire au projet.
