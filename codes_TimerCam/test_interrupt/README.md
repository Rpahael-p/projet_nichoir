# Test d'interruption

Ce code permet de tester le montage connecter à la pin de la LED (Transistor et bouton).

Si sur les 10 premières secondes, le bouton est allumé alors il change de mode de fonctionnement. Dans le code final, il lancera l'AP pour configurer la TimerCam.

Sinon, il fonctionnement "normalement", il fait clignoter la LED.
